SFX_HTTP
A Single File HTTP Server
Built by Addie Morrison, for CS464

High Performance!
Or at least would be in 1991!
Through its use of worker processes and async I/O, SFX_HTTP can serve a semi-reasonable number of connections per second!
Supports All Modern MIME Types!
